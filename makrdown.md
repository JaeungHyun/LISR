## DASANDATA MarkDown CheetSheet
|구분|문법|결과|
|---|---|---|
| 제목1단계 | \# 제목1 | <h1> 제목1 </h1> |
| 제목3단계 | \### 제목3 | <h3> 제목3 </h3> |
| 키보드 표시  |`<kbd>Ctrl+Shift+F1</kbd>` | <kbd>Ctrl+Shift+F1</kbd> |
| 코드삽입(짧게)| \`shtudown -r now`      | `shutdown -r now` |
| 코드삽입(줄바꿈)| \``` vi /etc/hosts  ```  | ``` vi /etc/hosts ``` |
